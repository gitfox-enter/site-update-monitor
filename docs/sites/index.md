# 支持的站点

RSSForge 目前配置了 **42 个** 订阅源，涵盖线报、购物比价、软件资源和社区论坛。

> 所有站点均由 GitHub Actions 自动定时抓取，数据每 15 分钟至 8 小时更新一次。
> 站点图标在首次运行后自动从各网站获取并缓存，无需依赖外部服务。

## 批量订阅（OPML）

不想逐个添加？下载 OPML 文件一次性导入所有站点：

| 文件 | 包含站点 |
|------|---------|
| [opml.xml](https://gitfox-enter.github.io/RSSForge/opml.xml) | 全部站点 |

::: tip 如何导入 OPML
大多数 RSS 阅读器都支持 OPML 导入。将上面的文件下载后，在阅读器中选择「导入 OPML」即可批量添加订阅。详见 [OPML 批量订阅](/feeds/opml)。
:::

## 更新频率说明

| 频率 | 适用场景 | 对应站点 |
|------|---------|---------|
| 每 15~20 分钟 | 高频更新，适合抢线报、秒杀信息 | 线报酷、线报ICU、专业线报、超级线报、汇发部、爱Q社区 等 |
| 每 30 分钟 | 中高频更新 | H6线报、天天赚、网赚、羊毛线报 等 |
| 每 1~2 小时 | 常规更新 | 白菜哦、拔草哦、慢慢买、豆瓣小组 等 |
| 每 4~8 小时 | 低频更新，适合内容变化慢的站点 | 反斗限免、423Down、果核剥壳、异次元软件 等 |

::: info 夜间降频
为节省 GitHub Actions 额度，所有站点在 **23:00 ~ 07:00（北京时间）** 期间自动降低抓取频率（间隔 ×3）。详见 [更新频率调度](/config/schedule)。
:::

::: info 图标说明
站点图标在首次���行 GitHub Actions 后自动从各网站获取并缓存到 `public/icons/` 目录，无需依赖 Google 等外部图标服务。如果某个站点的图标暂时缺失，系统会自动使用 DuckDuckGo 或 Icon.horse 作为备选，最终兜底为带首字母的 SVG 占位图。
:::

## 如何订阅

点击上表中「订阅」链接，复制地址后添加到你的 RSS 阅读器即可。不确定用哪个阅读器？查看 [阅读器推荐](/feeds/readers)。
