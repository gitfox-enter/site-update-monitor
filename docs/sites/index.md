## 批量订阅（OPML）

不想逐个添加？下载 OPML 文件一次性导入所有站点：

| 文件 | 包含站点 |
|------|---------|
| [opml.xml](https://gitfox-enter.github.io/RSSForge/opml.xml) | 全部站点 |
| [opml-线报站.xml](https://gitfox-enter.github.io/RSSForge/opml-线报站.xml) | 线报类 |
| [opml-购物比价.xml](https://gitfox-enter.github.io/RSSForge/opml-购物比价.xml) | 购物比价类 |
| [opml-软件站.xml](https://gitfox-enter.github.io/RSSForge/opml-软件站.xml) | 软件资源类 |
| [opml-论坛.xml](https://gitfox-enter.github.io/RSSForge/opml-论坛.xml) | 论坛类 |

::: tip 如何导入 OPML
大多数 RSS 阅读器都支持 OPML 导入。将上面的文件下载后，在阅读器中选择「导入 OPML」即可批量添加订阅。详见 [OPML 批量订阅](/feeds/opml)。
:::

## 更新频率说明

| 频率 | 适用场景 | 对应站点 |
|------|---------|---------|
| 每 15~20 分钟 | 高频更新，适合抢线报、秒杀信息 | 线报酷、线报ICU、专业线报 等 |
| 每 30 分钟 | 中高频更新 | H6线报、天天赚、网赚 等 |
| 每 1~2 小时 | 常规更新 | 白菜哦、拔草哦、慢慢买 等 |
| 每 4~6 小时 | 低频更新，适合内容变化慢的站点 | 反斗限免、423Down |

::: info 夜间降频
为节省 GitHub Actions 额度，所有站点在 **23:00 ~ 07:00（北京时间）** 期间自动降低抓取频率（间隔 ×3）。详见 [更新频率调度](/config/schedule)。
:::

## 如何订阅

点击上表中「订阅」链接，复制地址后添加到你的 RSS 阅读器即可。不确定用哪个阅读器？查看 [阅读器推荐](/feeds/readers)。
