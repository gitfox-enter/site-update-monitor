# RSS Reader Recommendations

Once you have feed sources generated by RSSForge, you will need a good RSS reader to enjoy them.

## Cross-Platform Readers

### [Folo](https://folo.io) - Recommended

> The reader officially recommended by RSSHub, enhanced with AI and purpose-built for RSSHub.

- Supports RSSHub route format
- Supports RSSForge standalone feeds
- AI-powered subscription recommendations
- Available on iOS / Android / Web
- Free + paid tiers

### [Inoreader](https://inoreader.com)

A well-established online RSS reader with a rich feature set.

- Cloud-synced across devices
- OPML import and export support
- Rule-based filtering and automation
- Free tier is sufficient for daily use
- Free + paid plans (starting at ~$10/year)

### [Feedly](https://feedly.com)

The most popular online RSS aggregator reader.

- Clean, polished interface
- Community sharing features
- High brand trust and reliability
- Free tier has ads and article limits
- Free + Pro plans

## Mac / iOS Native Readers

### [Reeder](https://reederapp.com) - Best for Mac/iOS

- Beautifully designed with an outstanding reading experience
- Supports RSS / Atom / JSON Feed
- Compatible with Fever, PocketBook, and other backends
- iCloud sync
- One-time purchase (Mac / iOS)

### [NetNewsWire](https://netnewswire.com)

A free, open-source reader for Mac and iOS.

- Completely free and open-source
- Clean, minimal interface
- Supports Feedbin, NewsBlur, and other backends
- Optimized for Apple Silicon
- Free

## Self-Hosted Readers

### [Miniflux](https://miniflux.app)

A lightweight self-hosted reader, deployable with a single Docker Compose file.

```yaml
version: '3'
services:
  miniflux:
    image: miniflux/miniflux:latest
    ports:
      - "8080:8080"
    environment:
      - DATABASE_URL=postgres://miniflux:secret@db/miniflux?sslmode=disable
      - CREATE_ADMIN=true
      - ADMIN_USERNAME=admin
      - ADMIN_PASSWORD=your-password
    depends_on:
      - db
  db:
    image: postgres:16
    environment:
      - POSTGRES_USER=miniflux
      - POSTGRES_PASSWORD=secret
      - POSTGRES_DB=miniflux
```

### [FreshRSS](https://freshrss.org)

A feature-rich self-hosted reader.

- Web interface with multi-user support
- Rich extension ecosystem
- OPML subscription import
- Easy Docker deployment

## Command-Line Readers

### [Newsboat](http://newsboat.org)

An RSS reader for the Linux/Mac terminal -- pure minimalism.

```bash
# Ubuntu/Debian
sudo apt install newsboat

# macOS
brew install newsboat

# Configuration
echo "https://USERNAME.github.io/RSSForge/opml.xml" >> ~/.newsboat/urls
newsboat
```

## How to Choose

| Use Case | Recommendation |
|----------|----------------|
| Best reading experience | Reeder (one-time purchase) |
| AI-assisted features | Folo |
| Multi-device sync, free tier | Inoreader |
| Mac/iOS focused, free | NetNewsWire |
| Self-hosted, Docker deployment | Miniflux |
| Terminal enthusiasts | Newsboat |
