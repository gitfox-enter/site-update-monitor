::: tip How to Import OPML
Most RSS readers support OPML import. Download the file above, then select "Import OPML" in your reader to bulk-add subscriptions. See [OPML Bulk Subscription](/en/feeds/opml) for details.
:::

## Update Frequency Guide

| Frequency | Use Case | Sites |
|-----------|----------|-------|
| Every 15-20 min | High-frequency updates, ideal for flash deals and limited-time offers | 线报酷 (XianBaoKu), 线报ICU, 专业线报 (Zhuanye Xianbao), etc. |
| Every 30 min | Medium-high frequency | H6线报 (H6 Xianbao), 天天赚 (TianTianZhuan), 网赚 (WangZhuan), etc. |
| Every 1-2 hours | Standard updates | 白菜哦 (BaiCaiO), 拔草哦 (BaCaoO), 慢慢买 (ManManMai), etc. |
| Every 4-6 hours | Low-frequency updates, for sites with slower content changes | 反斗限免 (FanDou Xianmian), 423Down |

::: info Nighttime Throttling
To conserve GitHub Actions quota, all sites automatically reduce crawl frequency during **23:00 - 07:00 (Beijing Time)** with intervals multiplied by 3x. See [Update Schedule Configuration](/en/config/schedule) for details.
:::

## How to Subscribe

Click the "Subscribe" link in the table above, copy the URL, and add it to your RSS reader. Not sure which reader to use? Check out [Reader Recommendations](/en/feeds/readers).
