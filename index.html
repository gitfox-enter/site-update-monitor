Everything is RSSible

基于 GitHub Actions 的免费 RSS 订阅源生成器。
给任何没有 RSS 的网站生成订阅源，无需服务器、零成本、可持续更新。

  • https://github.com/gitfox-enter/RSSForge
  • https://github.com/gitfox-enter/RSSForge/actions
🔧

零服务器成本

直接利用 GitHub Actions 免费计算资源，24 小时自动运行，完全零成本。

📡

RSS / OPML 输出

每站独立 feed + OPML 分类订阅包，导入任意 RSS 阅读器。

⚡

智能调度

每站独立更新间隔（15 分钟 ~ 8 小时），夜间自动降频，避免无效请求。

🔄

自动去重

MD5 指纹 + URL 去重 + 模糊标题合并，7 天滚动保留，订阅不重复。

📡 可订阅的订阅源

点击「订阅」获取该站点的 RSS 链接，复制到 RSS 阅读器中添加。

🚀 如何使用

1

Fork 本项目

点击右上角 Fork 按钮，一键复制到你的 GitHub 账号。

2

启用 GitHub Pages

Settings → Pages → Source 选 Deploy from a branch，分支选 gh-pages。

3

修改 sites.yaml

编辑 sites.yaml，添加你想监控的网站 URL 和抓取规则。

4

订阅 RSS

下载 opml.xml，粘贴到任意 RSS 阅读器（Reeder / FeedMe / inoreader 等）。
