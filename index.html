🚀 如何使用

1

Fork 本项目

点击右上角 Fork 按钮，一键复制到你的 GitHub 账号。

2

启用 GitHub Pages

在仓库 Settings → Pages → Source 选 Deploy from a branch，分支选 gh-pages（无则新建）。

3

修改 sites.yaml

编辑 sites.yaml，添加你想监控的网站 URL 和抓取规则。

4

订阅 RSS

在 feed.xml 复制链接，粘贴到任意 RSS 阅读器（Reeder / FeedMe / inoreader 等）。
